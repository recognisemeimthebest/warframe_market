## 1. 프로젝트 개요

### 1.1 목적
워프레임 게임의 아이템 시세를 실시간으로 추적하고, 웹 채팅 인터페이스에서 자연어로 가격 조회 및 급등 알림을 제공하는 AI 챗봇.

### 1.2 핵심 가치
- **24시간 무중단** — 라즈베리파이에서 상시 구동
- **자연어 대화** — 명령어가 아닌 자연스러운 한국어 대화
- **시세 감시** — 주식 시장처럼 가격 급등/급락 자동 알림
- **워프레임 지식** — 위키 기반 게임 정보 대화
- **접근성** — 스마트폰 브라우저에서 바로 접속

### 1.3 사용자
- 소규모 그룹 (약 10명)
- 워프레임 한국 서버 플레이어
- 스마트폰으로 주로 접속

## 3. 기능 명세

### 3.1 자연어 가격 조회

**설명**: 사용자가 채팅창에 자연어로 아이템 가격을 물어보면 warframe.market에서 현재 시세를 조회하여 답변.

**예시 대화**:
```
유저: 레진 프라임 세트 지금 얼마야?
봇:  레진 프라임 세트 현재 시세:
     - 판매 최저가: 45p
     - 구매 최고가: 40p
     - 48시간 평균: 42p
     - 거래량: 하루 23건
```

**동작 흐름**:
1. 유저 메시지 (WebSocket) → Gemma가 의도 파악 + 아이템명 추출
2. 아이템명 → warframe.market API 조회
3. API 결과 → Gemma가 자연어 응답 생성

### 3.2 시세 급등/급락 알림

**설명**: 주기적으로 주요 아이템 시세를 폴링하여, 급격한 가격 변동 감지 시 웹 채팅에 자동 알림.

**알림 조건** (초안, 튜닝 필요):
- 1시간 내 가격 변동률 ≥ 20%
- 24시간 평균 대비 ≥ 30% 이탈
- 거래량 급증 (평균 대비 3배 이상)

**알림 형식**:
```
📈 시세 급등 알림!
아이템: 레진 프라임 세트
현재가: 85p (1시간 전 45p, +88.9%)
24시간 평균: 42p
```

**폴링 주기**: 5분 (warframe.market API 제한 고려)

### 3.3 워프레임 지식 대화

**설명**: 워프레임 위키 데이터를 기반으로 게임 관련 질문에 답변. 워프레임 주제만 응답하고 다른 주제는 정중히 거절.

**범위**:
- 워프레임/무기/모드 정보
- 파밍 위치, 드롭 확률
- 빌드 기본 정보
- 게임 메카닉 설명

### 3.4 대화 범위 제한

**규칙**:
- 워프레임 관련 대화만 허용
- 관련 없는 질문 → "나는 워프레임 전문 봇이야! 워프레임 관련 질문을 해줘 🎮"
- 부적절한 내용 → 무시 또는 경고

### 3.5 웹 인터페이스

**설명**: 스마트폰 브라우저 최적화 채팅 UI.

**기능**:
- 실시간 채팅 (WebSocket)
- 시세 조회 결과 카드 형태 표시
- 모바일 반응형 레이아웃
- 인증 없이 접속 (소규모 그룹용)

## 4. 데이터 소스

### 4.1 warframe.market API
- **v2 엔드포인트**: `https://api.warframe.market/v2/`
  - `GET /items` — 전체 아이템 목록 (i18n 포함)
  - `GET /orders/item/{slug}` — 아이템 주문 목록
- **v1 엔드포인트**: `https://api.warframe.market/v1/`
  - `GET /items/{url_name}/statistics` — 시세 통계
- **제한**: 초당 3회
- **인증**: 공개 API (키 불필요, 일부 기능은 JWT 필요)

### 4.2 Warframe Wiki (Fandom API)
- **엔드포인트**: `https://warframe.fandom.com/api.php`
- **용도**: 게임 메카닉, 워프레임/무기 상세 정보
- **방식**: 위키 페이지 파싱 + 로컬 캐시

### 4.3 Warframe Public Export
- **엔드포인트**: `http://content.warframe.com/PublicExport/`
- **용도**: 공식 게임 데이터 (아이템, 드롭 테이블 등)
- **방식**: 주기적 동기화 → 로컬 JSON 저장

## 6. 제약 사항 및 리스크

| 리스크 | 영향 | 대응 |
|--------|------|------|
| Q8 응답 속도가 느림 | 대화 UX 저하 | Q4로 폴백 |
| warframe.market API 변경/다운 | 시세 기능 마비 | 캐시된 데이터로 폴백, 에러 알림 |
| Pi 5 메모리 부족 (동시 요청 시) | 크래시 | 요청 큐잉, 동시 추론 제한 (1개) |
| LLM 아이템명 오인식 | 잘못된 시세 제공 | 퍼지 매칭 + 확인 질문 |
| 위키 데이터 최신성 | 구버전 정보 제공 | 주기적 캐시 갱신 (일 1회) |
| 라즈베리파이 발열 | 성능 저하/셧다운 | 쿨러 필수, 온도 모니터링 |
| 외부 접속 보안 | 무단 접근 | 포트포워딩 또는 Tailscale VPN |
