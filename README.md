# 오디스 프라임

워프레임 관련 앱들을 쓰면서 불편했던 점이나, 이런 기능이 있었으면 좋겠다 싶었던 것들을 직접 만들어본 웹 앱입니다.

## 설치 방법

브라우저에서 접속한 뒤 홈 화면에 추가하면 앱처럼 사용할 수 있습니다.

**삼성 브라우저 (갤럭시)**
오른쪽 아래 메뉴(≡) → 추가 → 홈 화면 추가

**Chrome**
오른쪽 위 메뉴(⋮) → 홈 화면에 추가

**Safari (아이폰)**
하단 공유 버튼(↑) → 홈 화면에 추가

---

## 기능 소개

### 시세 조회
채팅창에 아이템 이름을 입력하면 실시간 최저가, 평균가, 48시간 가격 변동을 보여줍니다.
한글로 입력해도 영어로 자동 변환되어 마켓에서 조회됩니다.
다만 한글 변환이 아직 완벽하지는 않아서 영어로 검색하는 것을 추천드리며, 한글 지원은 계속 개선해 나갈 예정입니다.

### 시세 감시
사고 싶은 물건이 지금은 너무 비쌀 때, 원하는 가격을 설정해두면 마켓 최저가가 그 가격 이하로 내려왔을 때 알림을 보내줍니다.

### 급등 알림
1일, 1주, 1달 단위로 가격이 급등한 아이템들을 한눈에 볼 수 있습니다.

### 거래소
채팅방 사람들끼리 직접 거래할 수 있는 내수 거래소입니다.
사용하려면 저에게 승인 요청을 해주세요. 통제 목적이 아니라, 이 채팅방 멤버들만 이용할 수 있도록 하기 위함입니다.

### 월드 상태
- **균열 / 중재** — 본인이 원하는 미션 타입의 균열이 열리면 알림을 받을 수 있습니다.
- **침공** — 필드론, 인젝터 같은 소재는 제외하고, 무기 부품이나 오로킨 카탈리스트 등 가치 있는 보상이 뜨면 알림을 보내줍니다.

### 파밍 가이드
파밍하려는 아이템을 검색하면 파밍 방법을 알려줍니다.
한글, 영어 모두 검색 가능하지만 영어 사용을 추천드립니다.
현재는 간략한 정보만 제공되며, 업데이트하면서 점점 개선해 나갈 예정입니다.

### 모딩 공유
모딩 빌드를 채팅방에 올리면 나중에 다시 공유해달라고 하거나, 매번 사진을 찍어야 하는 번거로움이 있잖아요.
여기에 올려두면 언제든 다시 볼 수 있습니다. 간단한 메모도 함께 남길 수 있습니다.

### 설정
톱니바퀴 버튼에서 알림 설정, 다크/라이트 모드 전환, 색상 테마를 고를 수 있습니다.
