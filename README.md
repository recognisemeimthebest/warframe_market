# 오디스 프라임

워프레임 아이템 시세를 실시간으로 조회하고, 가격 알림을 받을 수 있는 웹 앱입니다.

## 사용 방법

브라우저에서 접속하면 바로 사용할 수 있습니다.
설정(톱니바퀴) → **홈 화면에 추가**를 누르면 앱처럼 설치됩니다.

## 기능

### 시세 조회
한국어 또는 영어로 아이템을 검색하면 실시간 최저가, 평균가, 48시간 가격 변동을 보여줍니다.
AI 챗봇에게 자연어로 질문할 수도 있습니다.

### 시세 감시
아이템에 목표 가격을 설정해두면, 해당 가격에 도달했을 때 자동으로 알림을 보내줍니다.

### 급등 알림
마켓 전체 아이템의 가격 급등/급락을 자동 감지하여 푸시 알림으로 알려줍니다.

### 거래소
현재 온라인인 판매자/구매자의 주문 목록을 실시간으로 확인할 수 있습니다.

### 월드 상태
시터스, 포츈아, 데이모스의 시간대와 균열, 침공, 중재 등 실시간 게임 이벤트를 표시합니다.
원하는 조건을 설정하면 해당 이벤트가 시작될 때 알림을 받을 수 있습니다.

### 파밍 가이드
아이템별 파밍 위치와 방법을 안내합니다.

### 모딩 공유
무기/프레임 모딩 빌드를 이미지로 올리고 다른 사용자들과 공유할 수 있습니다.

## 테마

설정에서 다크/라이트 모드 전환 및 5가지 색상 팔레트를 선택할 수 있습니다.
